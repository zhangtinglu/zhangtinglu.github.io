#

MAC上使用MWeb写博客自动同步印象笔记、博客园、github、csdn平台

发表于 2019-10-12 | 更新于: 2019-10-12 |
字数统计: | 阅读时长 ≈
目标：
  • 本地写博客自动同步到印象笔记，github、csdn、cnblogs、jianshu。统一采用Markdown的格式进行记录。
  • 第一，本地写博客的工具。不过说，选择MWeb，好用稳定方便，不多安利了。
  • 第二，怎样自动同步，MWeb一键式，且看下文
实施步骤
  1. 本地写博客。使用MWeb，可以截图粘贴，不要太好用。现在MWeb是收费，说实话找个破解版还是挺不容易的，直接花点钱省点时间吧。
  2. 使用github账号搭建博客。网上教程颇多，不重复造轮子。我搭建出来的效果：https://zhangtinglu.github.io/
  3. 博客园，注册账号即可。样式一般，但是比较专注，推荐。
  4. csdn，样式做的挺好，就是感觉广告太多。直接注册账号即可。
  5. 简书，主要就是注册几个主流账号吧。但是这个支持的不是很好，pass了
